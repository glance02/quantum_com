# 量子隐形传态

量子隐形传态是量子信息论中最核心的协议之一，它允许在不直接传输量子比特的情况下，将一个未知量子态

$$
|\psi\rangle = \alpha |0\rangle + \beta |1\rangle
$$

从一个位置（Alice）“传送”到另一个位置（Bob），仅通过共享的贝尔态资源和经典通信实现。本题目要求基于 pyqpanda3（>=0.3.2 版本）完整实现该协议，并进行深入的保真度分析。**要求不得使用除 pyqpanda3 和标准库外的其他库，numpy 等随 pyqpanda3 自动安装的库除外。**

已知标准量子隐形传态协议流程如下：

（一）对量子比特 `q1`, `q2` 制备一对最大纠缠态

$$
|\psi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)
$$

作为共享纠缠资源，其中 `q1` 是 Alice 的纠缠比特，`q2` 是 Bob 的纠缠比特；

（二）发送方 Alice 把具有未知量子态 $|\psi\rangle$ 的量子比特 `q0` 和自己的纠缠比特 `q1` 执行联合测量；

（三）Alice 将经典测量结果发送给接收方 Bob；

（四）Bob 根据经典信息，对自己的纠缠比特执行相应的修正操作，恢复出原始未知量子态。

请完成以下编程任务：

## 1、贝尔态制备（4 分）

实现一个函数 `prepare_bell_state(q1, q2)`，制备出标准的贝尔态

$$
|\psi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)
$$

## 2、量子隐形传态线路构建（8 分）

实现一个函数 `teleportation_circuit(q0, q1, q2)`，构建完整的量子隐形传态线路，包含：

（1）对 `q0` 制备未知量子态

$$
|\psi\rangle = \cos(\pi / 6)|0\rangle + \sin(\pi / 6)|1\rangle
$$

要求使用 $R_y$ 旋转门实现；

（2）对 `q1`, `q2` 制备贝尔态

$$
|\psi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)
$$

（3）Alice 对 `q0`, `q1` 进行测量前的联合操作（CNOT + Hadamard）；

（4）Bob 根据经典测量结果对 `q2` 施加条件修正操作。

## 3、协议验证与结果分析（8 分）

完成量子线路运行与正确性验证，要求如下：

（1）运行第 2 问中的量子线路，获取最终量子比特 `q2` 的量子态信息（振幅或测量概率）；

（2）对比原始量子态和 Bob 恢复后的量子态，计算量子态保真度；

（3）简要分析实验结果，证明量子隐形传态协议成功实现。
