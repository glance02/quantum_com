# 第三题技术说明

## 问题分析

本任务基于 Diabetic Retinopathy Debrecen 表格数据完成二分类。约束的重点有三点：

1. 两个模型必须共用完全一致的经典预处理流程。
2. 分类核心必须来自含可训练参数的量子变分线路。
3. 轻量模型需要在量子比特数和量子参数数上都小于基准模型。

## 方案概述

代码实现包含两个模型：

| 指标 | 基准模型 | 轻量化模型 |
| --- | ---: | ---: |
| 量子比特数 | 8 | 4 |
| 量子线路可训练参数 | 80 | 24 |
| 变分层数 | 3 | 2 |
| 量子测量输出维度 | 40 | 20 |

两者使用相同预处理：

1. 在原始 16 个字段基础上加入 10 个固定、无标签、无训练参数的组合特征：微动脉瘤总负担、首尺度响应、首末尺度差、首尺度相对均值，渗出物总负担、后段负担、后段占比，微动脉瘤/渗出物负担比，黄斑视盘距离与视盘直径比，以及该解剖比值与动静脉分类的固定交互项。
2. 基于训练集统计量做 1%/99% 分位数截尾。
3. 使用训练集均值和标准差做 Z-score 标准化。
4. 将结果截断到 `[-3, 3]`，并线性映射到 `[0, pi]` 作为量子旋转角，减少正负标准差在余弦型测量中的折叠。

## 量子模型结构

每个模型由角度编码、数据再上传、可训练旋转、环形纠缠和 Pauli 期望测量组成。

每层结构：

1. 将经典特征编码到 `RY` / `RZ` 旋转门。
2. 数据上传使用固定跨步顺序读取特征，使不同量子比特和不同层覆盖原始特征与组合特征。
3. 每个量子比特施加 `RY(theta) -> RZ(theta) -> RX(theta)`，其中 `theta` 为可训练量子参数。
4. 使用环形 `CNOT(q, q+1)` 纠缠，奇偶层反向纠缠以增强表达能力。
5. 测量 `Z_i`、相邻 `Z_i Z_{i+1}`、`X_i`、`Y_i` 与相邻 `X_i X_{i+1}` 期望值。

基准模型在 3 层变分线路之后额外加入 8 个最终 `RY` 可训练旋转门，使量子参数数达到题目允许上限 80。

测量结果后接一层线性读出。基准模型经典读出参数为 `40 + 1 = 41`，轻量模型为 `20 + 1 = 21`，均低于题目限制的 50。

## 训练方式

`train.py` 只读取 `data/train.csv`，并使用多个固定分层划分做内部交叉验证。量子参数采用多随机初始化筛选和少量 SPSA 精修，读出层在当前量子特征上拟合二分类交叉熵，并在小网格中选择 L2 正则强度。最终保存前使用全量训练集重新拟合读出层。

训练完成后保存两个独立模型文件：

```text
task3/artifacts/baseline_model.npz
task3/artifacts/lightweight_model.npz
```

同时额外保留一个合并文件 `task3/artifacts/qml_artifacts.npz` 作为兼容备份。`test.py` 不依赖训练流程，优先加载两个独立模型文件，并使用 `pyvqnet` 的 `QuantumLayerV3` 和 `pyqpanda3` 构造量子线路完成评估。为降低有限 shots 采样带来的预测抖动，测试阶段对同一模型重复推理数次并平均概率后再做阈值分类。

## 实验结果

本机运行：

```bash
python train.py
python test.py
```

一次运行示例输出：

```text
33.10,0.70,0.65,8,80,4,24
```

字段含义：

```text
Score,Acc_B,Acc_L,Q_B,P_B,Q_L,P_L
```

## 压缩分析

量子比特压缩率：

```text
(8 - 4) / 8 = 0.50
```

参数压缩率：

```text
(80 - 24) / 80 = 0.70
```

综合压缩比：

```text
0.5 * (0.50 + 0.70) = 0.60
```

轻量模型以较少量子比特和参数量保持了接近基准模型的测试准确率。
